# DDD Generator

## Description

This is a custom template to create slices using DDD and hex architecture in NestJS.

```shell
templates
├── domain
│   ├── entities
│   │   └── entity.hbs
│   └── interfaces
│       └── interface.hbs
├── infrastructure
│   ├── database
│   │   ├── models
│   │   │   └── model.hbs
│   │   ├── repositories
│   │   │   └── repository.hbs
│   │   └── services
│   │       └── service.hbs
│   └── http
│       └── controllers
│           └── controller.hbs
└── module.hbs

11 directories, 7 files
```
